[LAYER 11] TASK
Author the four artefacts with these contracts:

(A) src/predict/handler.py (NEW WRAPPER LAMBDA)
    Event in: { booking_id, fare_amount, refund_lag_hours, prior_cancels,
                event_id?, corr_id?, ...passthrough envelope fields }
    Return:   { ...passthrough, risk_score, confidence, model_version,
                low_confidence: bool, model_latency_ms: int }
    Behavior:
      1. Read ENDPOINT_NAME + CONFIDENCE_THRESHOLD env vars
      2. Extract 4 features from event (support both flat + payload-nested shapes)
      3. boto3 client config: connect_timeout=5, read_timeout=35, retries=max_attempts:0
      4. invoke_endpoint with:
            ContentType='application/json'
            Accept='application/json'
            Body=json.dumps(payload).encode('utf-8')
      5. Exception mapping:
            (ReadTimeoutError, ConnectTimeoutError) → raise EndpointColdStartTimeout
            ServiceUnavailableException OR HTTP 503 → raise EndpointThrottleException

            ModelError (non-504) → raise PipelineError
            Other ClientError → raise PipelineError
      6. Body parsing 5-step:
            .read() → bytes
            .decode('utf-8') BEFORE json.loads
            Validate model_version field
            Coerce numerics
            Wrap parse errors as PipelineError, NOT ValueError

      8. EMF metrics: model_invoked, low_confidence, fallback_used,
                       cold_start_timeout, throttle_exception,
                       model_latency_ms, confidence
